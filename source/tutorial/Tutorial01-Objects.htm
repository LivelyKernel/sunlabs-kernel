Lively Kernel Tutorial: (1) Basic Object Types & Moving Objects

Like most desktop environments, Lively Kernel applications consist of objects that can be manipulated directly by the user. A Lively Kernel application can contain various types of objects. The simplest object types include structures such as rectangles, ellipses, lines and text. In the area below you see a few examples of the basic object types.
