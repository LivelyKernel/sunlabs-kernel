Lively Kernel Tutorial: (11) Worlds and Subworlds

Lively Kernel applications and widgets usually live in an environment known as the world. A world is a visual container that can contain various types of objects and applications simultaneously.
Each world can contain any number of subworlds, identified by spiderweb-like LinkMorph icons.
You can create new subworlds by choosing the "new object..." operation from the popup menu of the world (i.e., popup menu of the background of the display).

Notes:

  • By default, LinkMorph icons utilize a visual "fisheye" feature: the icon is automatically enlarged when the mouse approaches the icon.
  • The fisheye feature for LinkMorphs (and for any other visual object) can be turned off/on from the popup menu of the object.

Try it yourself! Try shift-clicking the subworld icon (LinkMorph) below. Come back from the subworld by shift-clicking the corresponding LinkMorph in the subworld.
Furthermore, try creating new subworlds yourself by using the "new object..." operation available in the popup menu of the world.
