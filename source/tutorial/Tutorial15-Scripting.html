All the objects in the JavaScript Live system can be scripted using timers. Timers are JavaScript functions that are invoked periodically for the given objects.
For instance, in the example below we are using timers to rotate a number of stars that have been defined as polygon shapes.
See the source code of this demo for details.

Notes:

  • All the usual object editing capabilities can be applied to timed objects. For instance, you can edit the text in the labels below, or reshape the polygons on the fly.
