Lively Kernel Tutorial: (5) Using Popup Menus

Each object has a popup menu that can be invoked by clicking on the object while keeping the Alt key down.
Note that the background of the Lively Kernel ("the world") also has its own popup menu.

Notes:

  • The top area of the popup menu displays the type of the object.

Try it yourself! Press the Alt key while clicking on an object or the background of the world.
